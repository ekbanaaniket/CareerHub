# EduPlatform Features Guide

## How to Use the New Detail Pages

This guide explains how institute owners and administrators can access and utilize the comprehensive detail pages for courses, students, and tests.

---

## Course Detail View

### Accessing Course Details
1. Navigate to the **Courses** section
2. Click the **View** button (eye icon) on any course
3. You'll be directed to the Course Detail page

### What You Can See

#### Course Overview
- **Course Name & Status**: Current status (active, inactive, draft, completed)
- **Description**: Full course overview
- **Course Handout**: Downloadable or viewable course materials
- **Duration & Timeline**: Start date, end date, and course duration
- **Modules**: Number of learning modules
- **Topics**: List of all topics covered

#### Course Statistics
- **Total Students**: Number of enrolled students
- **Lectures**: Total lecture count
- **Tests**: Total tests/exams count
- **Capacity**: Maximum students allowed

#### Three Information Tabs

**1. Enrolled Students Tab**
- View all students enrolled in the course
- See their email and phone contact information
- Track individual student progress with progress bars
- View enrollment date for each student
- Quickly identify at-risk students (low progress)

**2. Lectures Tab**
- List of all course lectures
- Lecture module assignment
- Duration of each lecture
- Status (Scheduled, Live, Recorded)
- Navigate to specific lectures for more details

**3. Tests Tab**
- All course tests and exams
- Test type (Quiz, Exam, Assignment)
- Maximum marks for each test
- Current status of tests
- Click to view test details and submissions

## Student Detail View

### Accessing Student Details
1. Go to **Students** section
2. Click the **View** icon (eye symbol) next to any student name
3. The Student Detail page loads

### Student Profile Section
- **Profile Avatar**: Visual identifier
- **Name, Email, Phone**: Contact information
- **Member Since**: Enrollment date
- **Status**: Current enrollment status

### Key Statistics Cards
- **Attendance %**: Overall attendance percentage
- **Overall Progress %**: Average progress across all courses
- **Total Courses**: Number of enrolled courses
- **Status**: Active, inactive, or suspended

### Five Information Tabs

**1. Courses Tab**
- All courses the student is enrolled in
- Progress bar for each course showing completion %
- Enrollment date for each course
- "Completed" badge for finished courses
- Allows tracking of student's academic journey

**2. Tests & Exams Tab**
- All test submissions by the student
- Score achieved vs maximum score
- Submitted date for each test
- Comprehensive test performance tracking
- Identify weak areas based on scores

**3. Attendance Tab**
- Complete attendance history
- Organized by date
- Shows attendance status (Present, Absent, Late, Excused)
- Course-specific attendance tracking
- Date-wise attendance records

**4. Fees Tab**
- All fee records and payments
- Fee type (Tuition, Exam, Lab, Library, etc.)
- Amount due for each fee
- Payment status (Paid, Pending, Overdue, Waived)
- Due dates and payment dates
- Financial overview at a glance

**5. Job Applications Tab**
- All job applications submitted by student
- Employer (company name)
- Job position title
- Application status (Applied, Shortlisted, Interviewing, Selected, Rejected)
- Application date
- Salary offers if any

## Test Detail View

### Accessing Test Details
1. Go to **Tests** section
2. Click **View** button on any test
3. Test Detail page opens

### Test Overview
- **Test Name**: Title of the test/exam
- **Test Type**: Quiz, Exam, or Assignment
- **Status**: Draft, Upcoming, In Progress, or Completed
- **Description**: Test objectives and details

### Test Configuration Details
- **Duration**: Time allocated (e.g., 2 hours)
- **Total Questions**: Number of questions in test
- **Max Marks**: Total marks possible
- **Passing Marks**: Minimum marks required to pass
- **Test Date & Time**: When the test is scheduled

### Performance Statistics
- **Total Submissions**: How many students submitted
- **Average Score**: Mean score of all submissions
- **Pass Rate**: Percentage of students who passed
- **Passed/Failed Count**: Number breakdown

### Three Information Tabs

**1. Submissions Tab**
- Student-wise test submissions
- Score achieved by each student
- Percentage and performance metrics
- Pass/Fail status for each student
- Date submitted
- Sortable and filterable table

**2. Questions Tab**
- All test questions displayed
- Question number and text
- Question type (MCQ, Short Answer, Essay)
- Marks allocated per question
- Answer options (for MCQ)
- Correct answer highlighted
- Useful for question review

**3. Analysis Tab**
- Average score across all submissions
- Pass rate percentage
- Number of students who passed
- Number of students who failed
- Visual statistics dashboard
- Helps identify test difficulty

## Updated Students Page

### New Icon-Based Actions
Previously, the Students page used a three-dot menu. Now it uses individual icon buttons:

#### Action Icons in Students Table

1. **View Icon (Eye)** - Primary Action
   - Displays student's comprehensive profile
   - Shows all academic and enrollment data
   - Most commonly used action

2. **Edit Icon (Pencil)** - For Authorized Users
   - Edit student information
   - Update contact details
   - Only visible if you have edit permission

3. **Delete Icon (Trash)** - For Authorized Users
   - Remove student from system
   - Shows confirmation dialog
   - Only visible if you have delete permission
   - Red color to indicate destructive action

### Benefits of Icon-Based UI
✓ Faster access to actions (no dropdown menu)
✓ More intuitive user interface
✓ Better mobile responsiveness
✓ Clearer visual hierarchy
✓ Reduced clicks for common actions

## Common Workflows

### Workflow 1: Monitor Course Progress
1. Open **Courses**
2. Click View on target course
3. Go to **Enrolled Students** tab
4. Identify students with low progress (< 50%)
5. Take action (send reminders, provide support)

### Workflow 2: Track Student Performance
1. Open **Students**
2. Click View icon on a student
3. Check the **Tests & Exams** tab
4. Review scores and identify weak subjects
5. Check **Attendance** tab for correlation
6. Provide targeted support

### Workflow 3: Analyze Test Results
1. Open **Tests**
2. Click View on completed test
3. Check overall **Statistics**
4. Review **Submissions** to identify struggling students
5. Go to **Questions** tab to review difficult questions
6. Adjust teaching approach if needed

### Workflow 4: Review Student's Complete Profile
1. Students page → Click View icon
2. **Courses Tab**: See all enrollments and progress
3. **Tests Tab**: Review academic performance
4. **Attendance**: Check engagement
5. **Fees Tab**: Verify payments
6. **Jobs Tab**: See career placement status
7. Make informed decisions based on holistic view

## Troubleshooting

### Issue: Page takes time to load
**Solution**: Data is cached for 5 minutes. Refresh after making changes.

### Issue: Can't see certain tabs
**Solution**: Check your permissions. Some tabs may require elevated privileges.

### Issue: Student data not updating
**Solution**: The page caches data. Refresh the page or wait 5 minutes for cache refresh.

### Issue: Numbers not matching
**Solution**: Make sure you're looking at the correct institute/course filters.
